## Getting Started

First, ensure that node and npm are up to date (at least Node 22.19 and npm 10.9)
Next, run `npm install` to update packages.
Then, run `npx prisma generate` to generate client side prisma code
Then execute `npm run dev` to run development build, and view the project on [http://localhost:3000](http://localhost:3000)

To execute test suite, run `npm run test`

See examples environment files .env.example and .env.test.example for examples of needed .env (for local build) or .env.test (for local test)
